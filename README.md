# Typola

Typola is a rebranded Markdown editor based on Typora 1.9.5, with integrated plugins.

## Features

- Full Markdown WYSIWYG editing
- GitHub Copilot AI integration
- Built-in terminal emulator
- Plugin system with various utilities

## Installation

Download `Typola-Setup-1.9.5.exe` and run it. Choose an installation directory when prompted.

## Version

- **App version**: 1.9.5
- **Electron**: 30.0.9
